This starter project gives you everything you need to start working on a new website, and nothing more.

This web page was also used to practice basic html syntax. A sample of the html elements used are included below:

div
A generic container used to group html elements together.
h1
A section header. h1 is the highest section level.
p
A paragraph. Typically a block of text.
aside
Used for sections that are indirectly related to the main content. Typically represented as sidebars.
